# Bus Ticket System

A full-stack bus ticket booking system built with:
- Node.js + Express
- MongoDB
- React (Vite)

## Features
- User authentication
- Bus & route management (admin)
- Schedule creation
- Seat selection & booking
- Booking history

## Setup
1. Clone the repo
2. Install dependencies in `backend` and `frontend`
3. Add `.env` files
4. Run backend and frontend
